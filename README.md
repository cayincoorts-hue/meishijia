# 美食家 🍜

> 基于地理位置的、社区众包的本地美食优惠地图。

你发现了一个外卖隐藏优惠券？分享到这里，让同城的朋友们也享受到。

## 这是什么

美食家是一个**非商业、开源、社区驱动**的本地美食优惠发现工具。

- 🚫 **不是返利App** — 没有佣金链接，没有CPS分成
- 🔓 **完全透明** — 表格公开，规则公开，排名只看点赞数
- 📍 **精准到街道** — 找到你身边真正可用的优惠
- 👥 **社区互助** — 每个人分享自己发现的券，大家一起省钱

## 和返利App有什么不同

| | 返利App | 美食家 |
|---|---|---|
| 赚钱方式 | 你下单，平台拿佣金 | 不赚钱 |
| 推荐逻辑 | 谁佣金高推荐谁 | 谁点赞多谁排前面 |
| 数据 | 收集你的全部行为 | 只要你填的信息 |
| 代码 | 闭源 | 开源 |

## 怎么用

### 找优惠
1. 打开 [美食家表格](https://docs.qq.com/smartsheet/DQ1htcExwWXJYUmZt?tab=t00i2h)
2. 找到你的城市
3. 切换到 `🔥 24小时热门` 视图
4. 挑一个优惠，去对应App下手

### 分享优惠
1. 打开表格链接 → 微信扫码登录
2. 新增一行
3. 填写：餐厅名、所在城市、区/街道、外卖平台、优惠内容、优惠类型、有效期、**截图（必填）**、使用方式
4. 完成！同城的人就能看到了

## 规则

- ✅ 截图必填，没截图的优惠不显示
- 👍 点赞越多排越靠前
- 🚩 举报 ≥3 条自动隐藏
- ⏰ 过期优惠自动标记
