# 美食家 初始运营备忘录

> 2026-05-31，Phase 1 冷启动指南

## 初始城市

从你所在的城市开始，单一城市验证模式。

## 前两周要做的事

### 每日
- 添加 2-3 条真实优惠（当成模板，示范正确格式）
- 每条必附截图，信息填全

### 每周
- 检查：有没有新条目？有没有人点赞？有没有垃圾内容？
- 清理：过期优惠标记、举报 ≥3 的下沉

## 传播渠道

| 渠道 | 第一次发 |
|------|---------|
| 朋友圈 | "做了一个开源的本地美食优惠地图，非返利，大家一起来补充" |
| 小红书 | 带截图 + 项目介绍 + GitHub 链接 |
| GitHub | 仓库本身可被搜索 |

## 是否成功（前4周信号）

- 有人新增条目（不是你自己加的）
- 有人点赞
- 有人提 Issue 或反馈

## 什么时候进 Phase 2

同时满足：
1. 有 ≥5 个非自己的活跃分享者
2. 单一城市表格自然增长
3. 有人提了第二个城市的需求
