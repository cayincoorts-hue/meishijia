### 产品定位

- 名称：美食家
- 定位：基于地理位置的、社区众包的、非商业驱动的本地美食优惠发现工具
- 核心差异：非CPS、非返利、非平台抽成，社区互助 + 规则透明 + 开源中立
- 竞品分析：返利App（券券熊/好省丫）、高德扫街榜都是佣金/平台驱动，美食家是社区协议层

### 权限

- 查看：任何人可访问（免登录）
- 编辑：登录用户可编辑（微信扫码）

### 防恶意机制

截图必填、点赞自然筛选、举报阈值隐藏、位置隔离、编辑历史可追溯

## 技术决策记录

1. 城市字段：从下拉单选改为文本自由输入（腾讯文档下拉不支持批量添加，且用户自由输入更灵活）
2. 状态筛选：最初设计用公式+筛选，但腾讯文档筛选只支持数字。最终简化为纯排序视图，过期/隐藏手动文字标记
3. Git 推送：HTTPS 交互登录失败 → SSH 未配置 → gh CLI 需要 read:org scope → 最终用 token 嵌入 URL 推送成功
4. Token：用户的 GitHub token 在会话中暴露（`ghp_X4S...`），用户表示不删
